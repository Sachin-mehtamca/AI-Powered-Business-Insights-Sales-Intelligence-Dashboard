SQL Query Highlight: Category Contribution

MySQL 
SELECT category,
       SUM(sales) AS total_sales,
       SUM(profit) AS total_profit,
       ROUND((SUM(profit)/SUM(sales))*100, 2) AS margin_pct
FROM orders
GROUP BY category
ORDER BY total_sales DESC;
